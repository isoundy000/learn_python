2.13版本更新说明 2019年9月26日
  • 1、成长基金上线
    • 将原等级奖励更新为成长基金，免费奖励通过购买基金可升级为巨额奖励。
  • 2、新增鱼罐厂
    • 所有玩家每天可前往罐头工厂竞猜罐头生产日期，猜中立即获得奖励；VIP3及以上玩家可制作个人罐头供其他玩家竞猜。
　 　 　 2.12版本更新说明 2019年9月12日
  • 1、彩虹鱼玩法优化
    • 捕获彩虹鱼后必定可以给转盘充能，使用技能可以增大捕获几率！
  • 2、新增活动：中秋摇钱树
    • 玩家们每天5点、11点、17点、23点可免费摇动1次摇钱树，获得随机奖励。充值每满10元也可摇奖1次，累计摇动次数排名前十的玩家将获得招财珠奖励！
  • 3、背包内招财珠的特定功能迁移至活动页面。
　 　 　 2.11版本更新说明 2019年9月10日
  • 1、增加退出渔场时可快速换桌和换场的功能
    • 总是频繁切换场次的玩家这下可方便了！
  • 2、渔场转盘抽奖竞猜优化
    • 渔场转盘抽奖后，若翻牌竞猜失败，新增可使用钻石重新竞猜的功能。
　 　 　 2.09版本更新说明 2019年8月30日
  • 1、新增渔场内转盘抽奖玩法
    • 转盘抽奖会通过捕鱼积攒进度开启，越高级的渔场，抽中的奖励越丰厚！奖励还有机会通过竞猜翻倍！
  • 2、新增活动：鸿运当头
    • 玩家们每天0点、12点、18点、21点可免费获得1次抽奖机会，每次抽奖会额外获得积分，每天0点根据积分排行可获得黄金招财珠奖励！另外，充值每满10元也可获得1次抽奖机会！
